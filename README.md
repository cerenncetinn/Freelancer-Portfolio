#  Frontend Portfolio Website

Bu proje, HTML, CSS ve temel JavaScript öğrenme sürecinde geliştirilmiş, modern standartlara uygun bir kişisel portföy web sitesi çalışmasıdır.  
Projenin temel amacı; modüler SCSS mimarisini kavramak, grid/flexbox gibi modern layout yapılarını uygulamak ve frontend geliştirme disiplinini pekiştirmektir.

---

##  Öne Çıkan Özellikler

- Responsive (Duyarlı) Tasarım: Mobil, tablet ve masaüstü cihazlarla tam uyumlu yapı.
- Modüler SCSS Mimarisi: Değişkenler, partials ve nesting kullanılarak oluşturulmuş ölçeklenebilir stil yönetimi.
- Semantik HTML5: SEO ve erişilebilirlik ilkelerine uygun etiket kullanımı.
- Interaktif Bileşenler: - Ana sayfada daktilo (typewriter) yazı efekti.
  - Referanslar bölümü için Owl Carousel slider yapısı.
  - Proje galerisi için Lightbox2 entegrasyonu.

---

##  Kullanılan Teknolojiler ve Kütüphaneler

- HTML5 & CSS3 / SCSS
- JavaScript & jQuery
- Font Awesome: İkon kütüphanesi.
- Animate.css: Giriş animasyonları.
- Owl Carousel: Dinamik slider kütüphanesi.
- Lightbox2: Resim galerisi modali.

## Proje Yapısı

Proje, geliştirme dosyaları (SCSS) ve yayına hazır dosyalar (Public) olarak organize edilmiştir:

```text
├── public/                 # Yayına hazır dosyalar
│   ├── index.html          # Ana sayfa
│   ├── about.html          # Hakkımda sayfası
│   ├── projects.html       # Projeler sayfası
│   ├── contact.html        # İletişim sayfası

├── scss/                   # Kaynak SASS dosyaları (Geliştirme süreci)
│   ├── _variables.scss     # Renk ve tema değişkenleri
│   ├── _header.scss        # Navigasyon ve header stilleri
│   └── main.scss           # Tüm modüllerin birleştiği ana dosya
└── README.md
